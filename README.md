CoinDesk
========

Thin Go wrapper for [CoinDesk](http://www.coindesk.com/)'s BPI (Bitcoin
Price Index) API.


## Install

    go get github.com/mvillalba/go-coindesk/bpi


## Usage

See bpidemo/main.go for usage examples.
